# Zhiwei Hu Personal Website

This repository contains the static files for:

https://kzhkoshii.github.io

## Main files

- `index.html`: Home page
- `about.html`: Education, honors, news, and CV download
- `publications.html`: Publications and selected conference abstracts
- `chinese.html`: Chinese profile page
- `styles.css`: Visual design and responsive layout
- `script.js`: Mobile navigation
- `robots.txt` and `sitemap.xml`: Search engine indexing support
- `assets/`: Images and downloadable PDF documents

## Publishing with GitHub Pages

1. Upload all files and folders in this package to the root of the `kzhkoshii.github.io` repository.
2. Open the repository on GitHub.
3. Go to `Settings` → `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select the `main` branch and the `/ (root)` folder.
6. Click `Save`.

The website should become available at `https://kzhkoshii.github.io` after GitHub finishes deployment.

## Updating the website

- Edit page text directly in the corresponding `.html` file.
- Replace `assets/documents/Zhiwei_Hu_CV.pdf` when updating the CV.
- Add a new publication by copying an existing `<li class="publication-item">...</li>` block in `publications.html`.
